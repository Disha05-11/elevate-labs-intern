# Task 2 - Analyze a Phishing Email Sample

## Objective
Identify phishing characteristics in a suspicious email sample.

## Tools Used
- Sample phishing email
- Manual analysis

## Phishing Indicators Found

- Suspicious sender address
- Email spoofing attempt
- Urgent language
- Threat-based messaging
- Suspicious link
- Generic greeting

## Outcome

Learned how phishing attacks use social engineering and email spoofing to trick users into revealing sensitive information.
